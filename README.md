# dotfiles

設定ファイルだとか設定スクリプトだとかその他色々なんでも集約したもの。

## サブディレクトリ
* `.bin` - お手製の実行可能スクリプトを手動で突っ込む。
* `.aliases` - エイリアス設定ファイルが自動で突っ込まれる。
* `.memorandum` - メモを手動で突っ込む。
* `.build` - 何かビルドしたりするときとかに自動で使われる。
* `etc` - なんでもあり。メモではないものを手動で突っ込む。
